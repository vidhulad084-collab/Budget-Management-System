# Budget Management System

## Overview
A Budget Management System developed using Java, HTML, CSS, JavaScript, and SQLite. It helps users manage income and expenses, track balances, and generate reports.

## Technologies Used
- Java
- HTML
- CSS
- JavaScript
- SQLite

## Features
- Add Income
- Add Expenses
- View Income and Expenses
- Calculate Current Balance
- Search Records
- Generate Reports
- Export Reports

## How to Run
1. Clone this repository.
2. Open the project in Eclipse.
3. Run the application.
